## 🚀 Tech Stack
* **Frontend:** React.js, Tailwind CSS (Responsive Design)
* **Backend:** Node.js, Express.js
* **Database & BaaS:** Supabase (PostgreSQL)
* **Real-time & Auth:** Supabase Realtime / Supabase Auth

## ✨ Key Features
* **Digital Menu:** Dynamic menu display based on real-time database updates.
* **QR Code Integration:** Unique identifier system for table-specific ordering.
* **Admin Dashboard:** Comprehensive interface for restaurant owners to manage menus, prices, and track incoming orders.
* **Order Workflow:** Automated status updates from "Pending" to "Served".

## ⚙️ Technical Highlights
* **Relational Database Design:** Structured PostgreSQL schema via Supabase for robust data integrity (users, menus, orders).
* **Real-time Subscriptions:** Ready to handle live order tracking directly from the database.
* **Responsive UI:** Mobile-first approach to ensure customers can order easily from any smartphone.
